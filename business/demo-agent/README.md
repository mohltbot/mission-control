# OpenClautomation Slack Assistant - WORKING DEMO

## What It Does
- Monitors Slack channels for @mentions
- Responds intelligently using local LLM (FREE)
- Can schedule meetings, create tasks, draft emails
- Learns from conversation history

## Deployment (5 minutes)
1. `npm install`
2. Set SLACK_TOKEN and OPENCLAW_KEY in .env
3. `node deploy.js`

## Cost
- **FREE** - Uses local MLX inference (no API costs)
- Can handle 1000s of requests/month at $0

## Perfect For
- Teams wanting AI assistant in Slack
- Automating repetitive responses
- Meeting scheduling
- Task management
