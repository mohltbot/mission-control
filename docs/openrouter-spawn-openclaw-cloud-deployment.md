## Why It Matters for Our Stack

### OpenClaw Debugger

Currently OpenClaw agents run locally or via our own deployment scripts. OpenRouter Spawn
offers:

- **One-command cloud deployment** — no manual server setup or VPS management
- **Model-agnostic** — switch between Claude, GPT, Llama, Gemini without code changes
- **Auto-scaling** — agents scale down to zero when idle, spin up on demand

### ArchTrack

ArchTrack's desktop tracker currently requires manual setup per employee machine. Spawn
could provide a lightweight cloud relay agent that:
- Receives data from the desktop tracker
- Runs triage/summarisation logic server-side
- Pushes to the dashboard without requiring a self-hosted VPS
