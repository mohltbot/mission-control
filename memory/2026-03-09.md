## 6. 🎨 ArchTrack UI Fixes — March 9, 2026 (Afternoon)

**Time:** 3:00 PM - 5:30 PM PST  
**Focus:** Dashboard UI improvements and bug fixes

### Issues Fixed

#### Mock Data Cleanup
- **Problem:** Dashboard showing 8,076 suspicious activities and fake data (YouTube, Netflix, Excel, etc.)
- **Root Cause:** Tracker falling back to `getMockData()` when `active-win` failed
- **Fix:** 
  - Removed `getMockData()` function entirely from tracker
  - Changed error handling to skip recording instead of generating fake data
  - Deleted 11,943 fake activities from database
  - Kept 981 real activities (Finder, Chrome, Safari)

#### Activity Classification
- **Problem:** Safari showing as "Other" instead of "Core Work"
- **Fix:** Added Safari to Core Work classification (since Mac mini is dedicated OpenClaw work)
- **Added patterns:** `openclaw`, `claw`, `mohltbot`, `mission-control`, `archtrack`, `arch-track`

#### Time Calculation Bugs
- **Problem:** "15h today" inflated hours, negative hours in reports
- **Fix:** Changed calculation to use time between unique activity changes (capped at 10 min)
- **Before:** Summed all `duration_seconds` (10 sec per check = inflated totals)
- **After:** Actual time span between activity changes

#### Suspicious Activity Count
- **Problem:** Dashboard showing 0 suspicious activities but Reports showing 687
- **Fix:** Removed filter that excluded "No input activity" entries
- **Now counting:** ALL `is_suspicious = 1` activities (idle time = suspicious)

#### Focus/Idle Time Display
- **Problem:** "0h Focus Time" and "0h Idle Time" despite data existing
- **Fix:** 
  - Frontend: Show minutes when < 1 hour, hours when >= 1 hour
  - Backend: Include idle time in "wasted time" calculation
  - Dashboard now shows: 1h Focus Time, 11h Idle/Wasted Time

#### Sidebar Navigation Contrast
- **Problem:** Inactive nav items hard to read on dark sidebar
- **Fix:** 
  - Changed color from `#bdc3c7` (gray) to `#ffffff` (pure white)
  - Added font weight 500 for better readability
  - Improved hover effect opacity (0.15 vs 0.1)
  - Active item: Solid `#34495e` background with blue border
