## 🔧 Reusable Workflow Templates

### Template 1: Lead → CRM Automation
**Trigger:** New form submission/email/website visitor  
**Actions:**
1. Enrich lead data (Clearbit/Apollo)
2. Score lead (AI/custom logic)
3. Create CRM contact (HubSpot/Salesforce)
4. Send Slack notification to sales team
5. Trigger email sequence if qualified

### Template 2: Customer Onboarding Sequence
**Trigger:** New customer signup/purchase  
**Actions:**
1. Create onboarding checklist in project tool
2. Send personalized welcome email
3. Schedule onboarding call (Calendly)
4. Add to customer success dashboard
5. Trigger 30-day check-in reminder

### Template 3: Support Ticket Triage
**Trigger:** New support email/chat  
**Actions:**
1. Classify urgency (AI sentiment analysis)
2. Search knowledge base for similar issues
3. Auto-respond with potential solution
4. Route to appropriate team member
5. Create ticket in project management tool

### Template 4: Data Sync Between Tools
**Trigger:** Record created/updated  
**Actions:**
1. Map fields between systems
2. Transform data formats
3. Sync to secondary system
4. Log sync activity
5. Alert on failures

### Template 5: Report Generation & Distribution
**Trigger:** Schedule (daily/weekly/monthly)  
**Actions:**
1. Pull data from analytics/database
2. Generate report (PDF/Excel)
3. Email to stakeholders
4. Post to Slack/Teams
5. Archive in cloud storage
